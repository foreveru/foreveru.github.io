robotframework -- 环境安装

发表于 2017-05-14 | 分类于 自动化测试

今天跟大家介绍一下自动化测试框架Robotframework的环境安装。本文以在python环境的机器上安装和使用RobotFramework为例。

  1. 安装python
    下载网址：https://www.python.org/downloads/
    下载完成后，设置环境变量:计算机>属性>高级系统设置>环境变量>系统变量>Path, 点击新建，加入${python的安装路径} 和 ${python的安装路径}\Scripts, 例如：D:\Program Files\Python, D:\Program Files\Python\Scripts\Scripts
    打开DOS，输入python，验证环境是否安装好；

  2. 安装pip
    cd 到${python目录}\Scripts, z执行easy_install.exe pip,
    之后会在Scripts文件夹下出现一系列与pip有关的文件，其中有pip.exe, 说明pip命令可以使用；

  3. 安装WxPython
    Wxpython是python非常有名的一个GUI库，因为ride是基于这个库开发的，所以这个必须安装，
    https://sourceforge.net/projects/wxpython/files/wxPython/2.8.12.1 下载对应的python版本的uinicode版本的WxPython

  4. 安装Robot Framework
    DOS中执行 pip install robotframework

  5. 安装robotframework-ride
    这是robotframework的图形化界面
    DOS中执行 pip install robotframework-ride

  6. 安装需要的一些library
    直接pip安装，例如：
    pip install robotframework-selenium2library
    pip install robotframework-ftplibrary
    pip install robotframework-archivelibrary

Refrences：http://www.51testing.com/html/54/152754-867431.html
